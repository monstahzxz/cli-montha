Usage: commands [options] [command]

Attendance Management System

Commands:

To add an User
  addUser              add a user                
  findUser             find a user
  updateUser           update a user
  removeUser           remove a user
  listUsers            list all users

To add a subject
  addSubject           add a subject
  findSubject          find a subject
  updateSubject        update a subject
  removeSubject        remove a subject
  listSubjects         list all the subjects

To add students
  addStudents          add all the student details
  findStudents         find the students of a semester
  deleteStudents       deletes students of a semester

Other  
  help                 display help for command
  version              display the version of application
  exit                 exit
