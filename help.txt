Usage: commands [options] [command]

Attendance Management System

Options:
  -V, --version        output the version number
  -h, --help           display help for command

Commands:

To add an User
  addUser|au           add a user                
  findUser|fu          find a user
  updateUser|uu        update a user
  removeUser|ru        remove a user
  listUser|lu          list all users

To add a subject
  addSubject|as        add a subject
  findSubject|fs       find a subject
  updateSubject|us     update a subject

To add students
  addStudents          add all the student details

Other  
  help                 display help for command
  exit                 exit
